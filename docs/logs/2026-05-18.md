## Notes

- Default flip: previously the day view and resource-day-vertical view hardcoded `[0, 15, 30, 45]`, so the de facto default was 15-min slots in those views. All other vertical views already defaulted to hour-only. With this change, every hour-resolution view defaults to clean hourly rendering. Consumers who want the old day-view behavior should pass `slotDuration={15}`.
- The set `15 | 30 | 60` excludes `45` because it does not tile an hour evenly (60/45 = 1.33). Other divisors of 60 (5, 10, 12, 20) are excluded for now because the issue and the user explicitly asked for quarter-hour granularity. Easy to broaden later.
- Internal type uses `number` rather than the union `15 | 30 | 60` so that downstream extension does not require touching the grid layer.
