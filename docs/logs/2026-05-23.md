- **[hooks / pr-post approval]**: Added a PreToolUse Bash hook (`.claude/hooks/check-pr-post-approval.sh`) that blocks public-facing GitHub posting commands (`gh api .../pulls/.../{comments,reviews} -X POST`, `gh pr {comment,review,create}`, `gh issue {comment,create}`) unless `.claude/state/pr-post-approved.flag` was touched within the last 30 seconds. Hook contract verified against https://code.claude.com/docs/en/hooks (exit code 2 with stderr blocks the call; `tool_input.command` is read from stdin via `jq`).
