ilamy Calendar is a modern, open-source React calendar component for building robust calendar experiences. Designed for flexibility and ease of use, it comes with production-ready functionality, full TypeScript support, and seamless integration with Tailwind CSS v4.

<Aside type="tip">
ilamy Calendar supports Tailwind CSS v4. See the [installation guide](/docs/getting-started/installation) for setup instructions.
</Aside>
